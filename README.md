# quantumsite
A tool that uses quantum-inspired optimization algorithms to plan optimal robot movement paths on a construction site, visualized in a web interface

# Installation

## Requirements
- Python 3.12+

## Install (standard)

```bash
pip install quantumsite
```

## Install (development)

Clone the repository and install in editable mode so changes to the source are reflected immediately without reinstalling:

```bash
git clone https://github.com/jr173555/quantumsite.git
cd quantumsite
pip install -e ".[dev]"
```

To also install test and documentation dependencies:

```bash
pip install -e ".[dev,test,docs]"
```
